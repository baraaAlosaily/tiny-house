# tiny-house

## Install express

npm install express

## Install nodemon 

npm install nodemon -D

## Install typescript ts-node

npm install typescript ts-node -D

## Create tsconfig.json 

```
{
    "compilerOptions": {
            "target": "es6",
            "module": "commonjs",
            "rootDir": "./src",
            "outDir": "./build",
            "esModuleInterop": true,
            "strict": true,
    }
}   
```

## Install @types/express && @types/node

npm install @types/express @types/node
